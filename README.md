These addons add tons of new content related to the Half-Life world, mainly NPCs but also weapons and props. It also adds miscellaneous entities to make your game experience better.
#
I've also uploaded the .fdg file so you can mount it and use it in your Hammer installation (only if you install the addons locally in your Garry's Mod)
#
vjguardfixim - Is referred to the Base CETS Addon
vjguardfiximdlc1 - Is referred to the Aliens Expansion
vjguardfiximdlc3 - Is referred to the Humans Expansion
aspma - Is referred to the Portal Expansion
vjguardfixim_resources - Is referred to the Resources and Misc. Expansion
